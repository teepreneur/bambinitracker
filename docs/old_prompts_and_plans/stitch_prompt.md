# Google Stitch Master Prompt for Bambini

Copy and paste the prompt below into **Google Stitch** to generate the interactive UI prototype.

---

### [PROMPT START]

Create a high-fidelity mobile app prototype for **Bambini**, a child development tracker for parents and preschools.

**Brand Identity & Colors:**
- **Primary:** Turquoise/Teal `#2CC5BD` (from logo)
- **Secondary/CTA:** Warm Orange `#F5A623` (from logo star)
- **Accents:** Leaf Green `#8DC63F`, Soft Purple `#A67BB5`, Golden Amber `#F5D547`
- **Background:** Soft Cream-White `#F8FBFC`
- **Design Style:** Warm, modern, playful but professional. Use rounded corners (16px), soft shadows, and Nunito/Inter typography.

**App Architecture & Navigation:**
Use a persistent bottom tab bar with 5 icons: Home (🏠), Explore (🧩), Growth (📊), Messages (💬), and Profile (👤).

**Required Screens to Generate:**

1. **Splash/Welcome:** Turquoise-to-teal gradient background. Large white "Bambini" logo with footprint icon. Orange "Get Started" button.
2. **Parent Dashboard:** 
   - A greeting "Good morning, Ama 👋".
   - A turquoise "Stage Banner" showing "Toddler (18-24m)" with an orange 60% progress ring.
   - A "Today's Activities" list with 3 cards (Color Sorting, Obstacle Course, Story Time) using domain-colored icons.
3. **Activity Detail:**
   - A header image space.
   - Title "Color Sorting Game" with a purple "Cognitive" badge.
   - Checklist-style instructions and materials list.
   - Floating orange button: "How did it go?".
4. **Observation Feedback (Bottom Sheet):**
   - emoji rating row (Mastered, Getting There, Needs Practice, Skipped).
   - Text input for notes.
   - Photo/Video attachment buttons.
   - Orange "Submit" button.
5. **Growth Dashboard:**
   - Profile avatar at top.
   - Tab switcher: [Chart | Milestones | Timeline].
   - "Chart" view shows a 5-axis Radar Spider Chart using the accent colors (green, purple, orange, turquoise, gold) for different developmental domains.
6. **Teacher Class View:**
   - Greeting "Hello, Ms. Abigail".
   - A 3x3 grid of child avatars, each with a colored progress ring (green for on-track, orange for mixed).
   - Turquoise floating action button (+) for quick logging.

Ensure the transitions between screens feel smooth and the domain colors are consistently applied across badges and charts.

### [PROMPT END]
